# SponsorExplorer

A modern rewrite of Lanrza's [SBBrowser][sbb] and with more features!

## Installation

Requires Node.js v20 and Yarn package manager - install dependencies with
Yarn and start local dev server:

```console
yarn
yarn dev
```
