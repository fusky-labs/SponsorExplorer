# SponsorExplorer

A frontend to explore submitted SponsorBlock segments, inspired by [SBBrowser][sbb].

## Installation

Requires Node.js v20 and Yarn package manager - install dependencies with
Yarn and start local dev server:

```console
yarn
yarn dev
```
